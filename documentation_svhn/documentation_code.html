Documentation of the code of the simple example

Prerequisites

  • Python (the code was tested using Python 2.7.9 and Python 3.5.2)
  • numpy (version >= 1.11.0)
  • matplotlib
  • pillow
  • scipy
  • six
  • glymur, see GlymurWebPage
  • ImageMagick, see ImageMagickWebPage

Main functionalities

  1. How to test the library that contains tool functions?

    Each function in this library can be tested by passing its name as the argument of the script "test_tools.py". For instance,

    python test_tools.py visualize_weights

  2. How to test the class `VariationalAutoencoder`?

    Two methods of class `VariationalAutoencoder` can be tested by passing their name as the argument of the script "test_vae.py".

    python test_vae.py backpropagation
    python test_vae.py training

  3. How to test the class `EntropyAutoencoder`?

    Five methods of class `EntropyAutoencoder` can be tested by passing their name as the argument of the script "test_eae.py".

    python test_eae.py backpropagation_eae_bw
    python test_eae.py backpropagation_fct
    python test_eae.py training_eae_bw
    python test_eae.py training_fct
    python test_eae.py weights_decay

  4. How to create the SVHN training set, the SVHN validation set and the SVHN test set?

    The script "creating_svhn.py" downloads "train_32x32.mat", "test_32x32.mat" and "extra_32x32.mat" from the SVHN webpage, see SVHNWebPage. Then, it preprocesses the SVHN digits stored in the three files to create the SVHN training set, the SVHN validation set and the SVHN test set.

    python creating_svhn.py

  5. How to train a variational autoencoder on the SVHN training set?

    With the default arguments, the script "training_vae_svhn.py" takes around 1 day to fully train the variational autoencoder using a Intel Core i7-6500U CPU.

    python training_vae_svhn.py

  6. How to assess a trained variational autoencoder in terms of dimensionality reduction on the SVHN test set?

    python reconstructing_vae_svhn.py

  7. How to train a rate-distortion optimized autoencoder, also called entrotpy autoencoder, on the SVHN training set?

    Below, the value of the quantization bin width at the beginning of the training is 1.0 and the scaling coefficient is 10.0.

    python training_eae_svhn.py 1.0 10.0

  8. How to compare several trained entropy autoencoders, JPEG, and JPEG2000 in terms of rate-distortion on the SVHN test set?

    python reconstructing_eae_svhn.py
